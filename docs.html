Commands

Administration Commands Arguments (* : optional) Description
tutorial none You will receive a quick tutorial from the bot.
help settings* The bot will send you the help menu / settings menu.
botstat none The bot will send you overall statistics about itself.
donate none The bot will send you the donation link.
invite none The bot will send you its invite link.
Duels Commands Arguments (* : optional) Description
duel mention someone Will send a duel request to the mentioned user.
accept none Accept a duel request.
Lootboxes Commands Arguments (* : optional) Description
lootbox limited*
rare*		 Will send a duel request to the mentioned user.
open limited*
rare*		 Accept a duel request.
Inventory Commands Arguments (* : optional) Description
inventory page number* View your list of characters.
select char number Select the character.
remove char number Remove the character from your inventory.
info char number* View the infos of your character.
bal none View how many Lien (â± ) you have.
me none View your informations.
Trade Commands Arguments (* : optional) Description
trade create none The bot will send you a link to create a trade.
trade cancel none The link that the bot gave you will be disabled.
trade accept trade id Accept a trade you received.
trade delete trade id
 Delete a trade you sent.
trade refuse trade id
 Refuse a trade you received.
Market Commands Arguments (* : optional) Description
market none The bot will send you the market's URL.
market add char number + price Add a character to the market with a price.
market remove market id Remove a character from the market.
Arena Commands Arguments (* : optional) Description
arena? none Check whether or not there's an arena in the server.
joinarena none Joins an arena.
Shop Commands Arguments (* : optional) Description
shop none View what's in the shop.
buy item number Buy an item from the shop.
Missions Commands Arguments (* : optional) Description
mission go none If a mission is available, your character will go.
mission end none If you have an ongoing mission, it'll stop.
dungeon none Launches a dungeon to get some extra loot.
Other Commands Arguments (* : optional) Description
daily none Claim your daily reward for voting on top.gg.
comp -c (name) + -l (level) + -v (value) + -r (rarity) View a character that would have the above stats.
